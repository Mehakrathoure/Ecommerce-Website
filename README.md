# 🛒 E-commerce Website (Frontend Project)

## 📌 Project Overview

This is a responsive **E-commerce website UI** built using HTML, CSS, JavaScript, and Bootstrap.
The project focuses on creating a modern and user-friendly shopping interface with product listings, navigation, and basic UI interactions.

---

## 🚀 Features

* 📱 Fully responsive design (mobile, tablet, desktop)
* 🧭 Navigation bar with search functionality
* 🎞️ Image carousel for banner/slider
* 🛍️ Product listing using cards
* 🛒 Add to Cart & Buy Now buttons (UI level)
* 🔗 Social media icons integration
* 📄 Footer section with company details

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **Bootstrap** – Responsive layout & components
* **JavaScript** – Basic interactivity
* **Font Awesome** – Icons

## 💡 Key Components
